Authentication

StreamlineVPN API uses JWT (JSON Web Tokens) for authentication. Include your token in the Authorization header.

Get Access Token

POST
/auth/login 
{
  "username": "your_username",
  "password": "your_password"
}
Returns: {"access_token": "jwt_token", "expires_in": 3600}

Using the Token

curl -H "Authorization: Bearer YOUR_JWT_TOKEN" \
     https://api.streamlinevpn.com/v2/health

Endpoints

Health Check

GET
/health

Check API server health and status.

{
  "status": "healthy",
  "version": "2.0.0",
  "timestamp": "2025-01-10T12:00:00Z"
}

Get Sources

GET
/api/v1/sources

Retrieve a list of configured VPN sources.

{
  "sources": [],
  "total": 0,
  "timestamp": "2025-01-10T12:00:00Z"
}

Get Configurations

GET
/api/v1/configurations

Retrieve processed VPN configurations.

{
  "configurations": [],
  "total": 0,
  "limit": 100,
  "offset": 0,
  "timestamp": "2025-01-10T12:00:00Z"
}

Start Processing Pipeline

POST
/api/v1/pipeline/run

Start a new VPN configuration processing pipeline.

{
  "config_path": "config/sources.yaml",
  "output_dir": "output",
  "formats": ["json", "clash"]
}
Returns: {"job_id": "job_123", "status": "accepted", ...}
