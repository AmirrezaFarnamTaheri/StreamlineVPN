Configuration Overview

StreamlineVPN uses YAML-based configuration files to manage VPN sources, processing pipelines, and output formats. This guide covers all configuration options and best practices. 

config/
├── sources.yaml          # VPN source definitions
├── processing.yaml       # Processing pipeline config
└── outputs.yaml          # Output format settings

Quick Start Configuration

# config/sources.yaml
sources:
  - name: "Premium Provider"
    url: "https://example.com/configs.txt"
    tier: "premium"
    enabled: true
    
processing:
  max_concurrent: 10
  quality_threshold: 0.7

output:
  formats: ["json", "clash"]
  directory: "output"
