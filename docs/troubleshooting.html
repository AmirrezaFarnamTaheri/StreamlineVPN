StreamlineVPN Troubleshooting Guide

Installation Issues

If you're having trouble installing StreamlineVPN:

  • Ensure Python 3.8+ is installed: python --version
  • Install dependencies: pip install -r requirements.txt
  • Check for permission issues on Windows/macOS

API Issues

Common API problems and solutions:

  • Check if the server is running: curl http://localhost:8080/health
  • Verify port availability (8080 for API, 8000 for web)
  • Check firewall settings

Performance Issues

If StreamlineVPN is running slowly:

  • Check Redis connection for caching
  • Monitor memory usage
  • Review source configuration for too many sources

Network Issues

Network-related troubleshooting:

  • Verify internet connectivity
  • Check proxy settings if applicable
  • Test source URLs manually

Diagnostics

Run these commands to diagnose issues:

  • python scripts/comprehensive_validator.py - Full project validation
  • python -m streamline_vpn --help - CLI help
  • Check logs in the logs/ directory
