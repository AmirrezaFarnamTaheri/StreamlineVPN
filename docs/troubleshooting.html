üîå API Connection Errors

Error: "Failed to load sources"

Cause: API endpoint is unreachable or returning errors

Solutions:

  • Check if the backend service is running
  • Verify API_BASE configuration in api-base.js
  • Check network connectivity and firewall settings
  • Review browser console for detailed error messages

Error: "Unexpected token '<'"

Cause: API is returning HTML instead of JSON (likely 404 or 500 error)

Solutions:

  • Verify the API endpoint URL is correct
  • Check if the backend server is properly configured
  • Ensure the API routes are properly defined
  • Check server logs for detailed error information

‚öôÔ∏è Configuration Problems

Issue: Invalid YAML Configuration

Symptoms: Configuration parsing errors, pipeline failures

Solutions:

  • Validate YAML syntax using online YAML validators
  • Check indentation (use spaces, not tabs)
  • Ensure all required fields are present
  • Review configuration schema documentation

Issue: Source URLs Not Accessible

Symptoms: Sources showing as offline, no configurations loaded

Solutions:

  • Test source URLs manually in browser
  • Check if sources require authentication
  • Verify network connectivity and proxy settings
  • Update source URLs if they've changed

‚ö° Performance Issues

Issue: Slow Processing

Symptoms: Long processing times, timeouts

Solutions:

  • Reduce max_concurrent_sources in configuration
  • Increase timeout values for slow sources
  • Filter out low-quality sources
  • Use caching to avoid reprocessing

Issue: High Memory Usage

Symptoms: System slowdown, out of memory errors

Solutions:

  • Reduce batch sizes in processing configuration
  • Enable garbage collection optimization
  • Limit the number of concurrent operations
  • Monitor memory usage with system tools

‚ùì Frequently Asked Questions

How do I get started with StreamlineVPN?

Start by reading our Quick Start Guide, then configure your sources in the Configuration Guide. Use the Control Panel to manage your VPN configurations.

What VPN protocols are supported?

StreamlineVPN supports VLESS, VMess, Trojan, Shadowsocks (SS), and ShadowsocksR (SSR) protocols. The system automatically detects and processes configurations for all supported protocols.

How does the quality prediction work?

Our ML-powered quality assessment uses LSTM neural networks to analyze latency, bandwidth, uptime, location, and protocol factors. Configurations are scored from 0.0 to 1.0, with higher scores indicating better quality.

Can I add custom VPN sources?

Yes! You can add custom sources by editing the sources.yaml configuration file. Sources can be URLs pointing to configuration files, APIs, or other data sources. See the Configuration Guide for details.

What output formats are available?

StreamlineVPN supports JSON, Clash, Sing-Box, and Base64 output formats. You can configure multiple formats in your processing pipeline and export configurations in any supported format.

How do I troubleshoot API connection issues?

Check the API Connection Errors section above. Common solutions include verifying the backend service is running, checking network connectivity, and reviewing the API_BASE configuration.
